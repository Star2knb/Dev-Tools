## Description

A simple tool for creating a readme.md file for your project, and also contains a dependency checker.

[This Readme file was created with it.]

[You can use it here](https://dev-tools-tau-black.vercel.app/)

## Features

- Easy to use fields for creating your README.md file
- Easily add badges
- Import your package.json file to check all your packages
- Button to update all packages to the latest

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```
